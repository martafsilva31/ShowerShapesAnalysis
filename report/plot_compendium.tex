\section{Initial Validation Plots}
\label{sec:validation}
% ============================================================================

These plots validate the cell-level recomputation of the three shower shape
variables ($R_\eta$, $R_\phi$, $w_{\eta\,2}$) against the stored values in
the ntuples. Each figure shows three normalised distributions:
\textbf{Unfudged/stored} (blue) — the original MC stored values;
\textbf{Fudged/stored} (red) — the ``fudged'' stored values; and
\textbf{Computed from cells} (green, dashed) — values recomputed from the
$7\times11$ Layer-2 cell energies. The lower panel displays the ratio to the
Unfudged baseline. Agreement between the blue and green curves confirms
correct cell-level recomputation.
